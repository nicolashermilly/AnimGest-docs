⚠️ Règles critiques — à ne jamais oublier
🐳 Docker Compose
  • Toujours cd deploy avant toute commande Docker
  • docker compose restart recrée depuis l'image — efface les patches manuels
  • Utiliser docker stop + docker cp + docker start pour préserver les fichiers injectés
  • Ne jamais utiliser inject_v4_final.js — SyntaxError fatal
🏗️ Build frontend
  • Toujours lancer .\COPIER_TOUTES_PAGES_v2.ps1 AVANT le build
  • Séquence : build frontend → up -d frontend → restart nginx
  • Ne jamais overwrite layout.tsx automatiquement
🗄️ BDD & API
  • Routes custom sans préfixe /api/ — nginx le strip
  • Utiliser mysql2/promise direct (pas DbService)
  • DbService inaccessible après app.listen()
  • Vérifier information_schema avant tout SQL supposé
💾 Backups
  • Backup propre : C:\Backup\micro_logiciel\20260315_0002
  • Corrompus à partir de 20260315_0042 — ne jamais utiliser
  • Tâche planifiée : PetSuite_AutoStart — démarrage 60s après login
