Généalogie élevage — conditionnelle

Le composant <GenealogieTree /> n'est visible que si :

  • Le métier ELEVEUR est activé dans les modules (vérifié via /api/parametres/modules)
  • L'utilisateur est sur /app/elevage/dashboard (le lien sidebar n'existe que pour ce métier)
  • Le toggle "Voir/Masquer généalogie" est activé (état local React)

L'arbre SVG représente : Luna × Arès → Portée B (sept 2023) → chiots vendus/garde + Portée O (mars 2026) → Thor/Freya réservés.
