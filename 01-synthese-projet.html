Nginx — Routing

Nginx agit comme reverse proxy et strip le préfixe /api/ avant de transmettre à NestJS.

# Nginx strip /api/ → routes internes sans préfixe
location /api/ {
    proxy_pass http://micro_api:8080/;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
🚨
Les routes internes NestJS n'ont PAS le préfixe /api/. Une route accessible via GET /api/clients est enregistrée en interne comme GET /clients.

Règle critique — Injection de routes API

🔴
RÈGLE ABSOLUE : Utiliser uniquement _h56.use() pour injecter des routes dans dist/main.js.
Les méthodes _h56.get(), _h56.post(), _h56.patch() retournent systématiquement 404.

Procédure d'injection de routes (inject_v3_force.js)

# Sauvegarder avant toute modification
$bk = "C:\Backup\micro_logiciel\$(Get-Date -Format 'yyyyMMdd_HHmm')"
New-Item -ItemType Directory -Force -Path $bk
docker cp micro_api:/app/dist/main.js "$bk\main.js"

# Copier et exécuter le script d'injection
docker cp "C:\Users\AdminPC\Downloads\inject_v3_force.js" micro_api:/tmp/
docker exec micro_api node /tmp/inject_v3_force.js
docker compose restart api

# Vérification
Start-Sleep 8
Invoke-WebRequest -Uri "http://localhost/api/ged-factures-impayees" -UseBasicParsing | Select StatusCode
# Doit retourner 200
⚠️
Les patches API sont volatils : ils sont perdus si l'image API est rebuildée. Toujours sauvegarder main.js avant toute opération.

GED — Validation inter-bases (Trigger)

Les triggers bi_ged_document_lien_check_objet et bu_ged_document_lien_check_objet vérifient l'existence de l'objet métier dans micro_logiciel avant insertion/modification d'un lien.

-- Exemple de vérification dans le trigger (BEFORE INSERT)
CASE v_code
  WHEN 'ANIMAL' THEN
    SELECT COUNT(*) INTO v_exists
    FROM micro_logiciel.animal
    WHERE ANI_ID = NEW.OBJ_ID;
  WHEN 'FACTURE' THEN
    SELECT COUNT(*) INTO v_exists
    FROM micro_logiciel.facture
    WHERE FAC_ID = NEW.OBJ_ID;
  -- ... 11 autres types ...
END CASE;

IF v_exists = 0 THEN
  SIGNAL SQLSTATE '45000'
    SET MESSAGE_TEXT = 'Objet métier introuvable pour ce lien GED';
END IF;
