1

ETAPE 1 — Configurer le profil naturopathe

  • → Paramètres → Mon métier : sélectionner Naturopathe
  • Espèces traitées : chiens, chats, chevaux, lapins, NAC
  • Types de prestations : bilan nutritionnel, consultation, suivi, atelier
  • Tarifs par durée et type de consultation (présentiel / distanciel)
2

ETAPE 2 — Créer le dossier patient

  • → Clients → Nouveau client + fiche animal détaillée
  • Antécédents : pathologies, allergies, traitements en cours, vaccins
  • Alimentation actuelle : type (croquettes, BARF, fait maison), marque, quantité
  • Mode de vie : activité, stress, environment, autres animaux
3

ETAPE 3 — Bilan nutritionnel initial

  • → Séances → Nouvelle séance : type Bilan nutritionnel
  • Analyse : carences potentielles, excès, déséquilibres
  • Evaluation état corporel (score BCS), pelage, digestion
  • Plan nutritionnel personnalisé : rations, compléments, transitions
4

ETAPE 4 — Recommandations et programme

  • → Programmes → Nouveau programme
  • Plan de complémentation : produits, dosages, durée, protocole de sevrage
  • Recettes de rations ménagères adaptées à l'animal
  • Envoyer via portail client : programme + recettes en PDF
💡 Coordonner toujours avec le vétérinaire traitant pour les animaux sous traitement médical.
5

ETAPE 5 — Vente de produits naturels

  • → Paramètres → Articles : créer les compléments, huiles, herbes
  • Stock avec seuil d'alerte et réapprovisionnement
  • Vente directe en consultation : article ajouté à la facture de séance
  • Suivi des ventes par produit pour optimiser les stocks
6

ETAPE 6 — Suivi et r&éévaluations

  • → Séances → Suivi CAC pour évaluer l'évolution
  • Bilan à 1 mois, 3 mois, 6 mois : adapter le programme
  • Photos avant/après (pelage, poids) dans la GED
  • Questionnaire de suivi envoyé automatiquement entre les consultations
7

ETAPE 7 — Ateliers et formations

  • → Séances de groupe : ateliers (introduction BARF, cuisine animale)
  • Capacité max, tarification différenciée
  • Supports de formation dans la GED, partageables aux participants
  • Devis groupe pour les ateliers en entreprise ou association
8

ETAPE 8 — Facturation et comptabilité

  • → Factures : séparation prestations (TVA 20%) et produits (taux selon nature)
  • Export TVA : attention aux taux variables selon les produits
  • Notes de frais : formations, ouvrages spécialisés, déplacements
  • Export FEC pour expert-comptable
