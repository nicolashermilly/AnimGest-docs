🛒 Guide Module Achats

Anciennement 49-achats-guide — renommé 52 (slot libre) pour éviter la collision avec 49-module-veterinaire

📌 Ce fichier documente l'utilisation du module achats d'un point de vue utilisateur. Pour la doc technique, voir 33-module-achats-complet.html.

📦 Flux de travail utilisateur

Créer une commande → Valider → Réceptionner (pointage BL) → Rapprocher facture fournisseur → Régler → Générer écriture comptable.

1. Nouvelle commande
Aller dans Achats > Nouvelle commande. Sélectionner le fournisseur, ajouter les lignes articles avec quantités et prix unitaires.
2. PDF & Email
Depuis la fiche commande : cliquer PDF (modal) ou Mail (envoi au fournisseur).
3. Réception
Onglet Bons de livraison : cocher les lignes reçues. Saisir quantités partielles si livraison incomplète.
4. Avoir fournisseur
Si retour marchandise : Bons de retour > Créer demande d'avoir. Impact automatique sur le stock.
